# Fixture Import UI

A web application for managing and importing fixtures into RightsLogic.

## Getting Started

Open `web/index.html` in your browser, or run a local dev server:

```bash
npx serve web
```

## Project Structure

```
web/
  index.html       — Main application (single-page app)
  css/styles.css   — Stylesheet
  js/app.js        — Application logic
```
